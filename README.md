# instagram-content-factory
Instagram投稿素材を自動生成するための作業用リポジトリ
## Codex運用ルール

CodexはInstagram投稿素材を作成するとき、必ず以下のファイルを参照する。

1. ブランド.md
2. アイデア/投稿アイデア.md
3. 既存の outputs/*.md

新しい投稿パックは `outputs/YYYY-MM-DD_テーマ名_post_pack.md` として作成する。

各投稿パックには必ず以下を含める。

- 投稿テーマ
- 投稿目的
- 想定ターゲット
- キャプション
- ハッシュタグ
- カルーセル構成
- 画像生成用プロンプト3案
- リール台本
- CTA
- 次回投稿へのつなぎ

トーンは `ブランド.md` の内容を優先する。
薬機法リスクのある断定表現は避ける。
## 参照優先順位

1. README.md（運用ルール）
2. ブランド.md
3. アイデア/投稿アイデア.md
4. templates/*.md
5. outputs/*.md
6. CodexはInstagram投稿素材を作成するとき、必ず以下のファイルを参照する。

1. ブランド.md
2. アイデア/投稿アイデア.md
3. カレンダー/投稿カレンダー.md
4. templates/*.md
5. outputs/*.md
既存の outputs/*.md は上書きしない。

新しい投稿は新しいファイルとして作成する。
迷った場合はブランドの世界観を優先し、
売り込みではなく価値提供を目的とした投稿を作成する。
