# instagram-content-factory

Instagram投稿素材を自動生成するための作業用リポジトリ。

## Codex運用ルール

CodexはInstagram投稿素材を作成するとき、必ずこのREADME.mdを最初に読み、以下の参照ファイルを確認する。

## 参照ファイル

1. ブランド.md
2. アイデア/投稿アイデア.md
3. カレンダー/投稿カレンダー.md
4. templates/*.md
5. 既存の outputs/*.md

## 参照優先順位

1. README.md（運用ルール）
2. ブランド.md（ブランドの世界観・トーン）
3. アイデア/投稿アイデア.md（投稿テーマ候補）
4. カレンダー/投稿カレンダー.md（投稿予定・ステータス）
5. templates/*.md（出力フォーマット）
6. outputs/*.md（過去投稿の参照）

## 出力ルール

新しい投稿パックは、以下の形式で作成する。

`outputs/YYYY-MM-DD_テーマ名_post_pack.md`

既存の `outputs/*.md` は上書きしない。  
新しい投稿は新しいファイルとして作成する。

## 投稿パックに必ず含める項目

- 投稿テーマ
- 投稿目的
- 想定ターゲット
- キャプション
- ハッシュタグ
- カルーセル構成
- 画像生成用プロンプト3案
- リール台本
- CTA
- 次回投稿へのつなぎ

## トーン＆表現ルール

トーンは `ブランド.md` の内容を優先する。  
迷った場合はブランドの世界観を優先し、売り込みではなく価値提供を目的とした投稿を作成する。

## 禁止事項

- 薬機法リスクのある断定表現は避ける
- 「治る」「改善する」「効果がある」と断定しない
- 不安を強く煽らない
- 他社や既存化粧品を否定しない
- エコを押しつける表現にしない
